# Duck Ponder System Images

This folder contains the images used by the Duck Ponder System to display information about the different duck types.

## Image Requirements:
- All images should be PNG format
- Ponder page images should be 176x100 pixels
- Use consistent style across all ponder images
- Recommended to use a slightly darker background with lighter elements for contrast

## Required Images:

### Base Ponder System:
- ponder_icon.png - 16x16 icon for hovering
- ponder_background.png - 256x180 background for all ponder screens
- ponder_loading_bar.png - 16x2 loading bar texture
- lightning.png - 16x32 lightning effect for the golden duck ritual

### Rubber Duck Pages:
- rubber_duck_page1.png - Introduction to rubber duck
- rubber_duck_page2.png - Crafting illustration
- rubber_duck_page3.png - Effects illustration

### Golden Rubber Duck Pages:
- ritual_page1.png - Introduction to golden rubber duck
- ritual_page2.png - Altar construction
- ritual_page3.png - Required ingredients
- ritual_page4.png - Activation process

### Gold Ore Rubber Duck Pages:
- gold_ore_duck_page1.png - Introduction to gold ore duck
- gold_ore_duck_page2.png - Mining chance illustration
- gold_ore_duck_page3.png - Effects illustration

## Adding New Duck Ponder Images:
1. Create your PNG images at 176x100 resolution
2. Place them in this folder with an appropriate naming scheme
3. Reference them in the corresponding Ponder screen class
4. Update the lang file with appropriate descriptions
